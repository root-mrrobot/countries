About Countries

Welcome to the Countries app! This app is designed to provide comprehensive information about countries around the world.

Technologies Used

  • Angular
  • HTML
  • CSS
  • TypeScript

Features

  • Detailed country information
  • Search functionality
  • Filtering by region and population

Data Source

The country data is fetched from the REST Countries API, which offers a wealth of information about countries worldwide. Visit the REST Countries website for more details.
